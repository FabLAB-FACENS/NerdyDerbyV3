# ESP32 LARGADA — Gateway Node

Este módulo atua como gateway entre os dispositivos e a aplicação.

## Funções

- receber dados do ESP32 CHEGADA via ESP-NOW
- conectar na rede Wi-Fi
- publicar dados no MQTT
- receber comandos do app
- controlar atuadores

---

## Comunicação

- **Recebe de:** ESP32 CHEGADA (ESP-NOW)
- **Envia para:** MQTT
- **Recebe de:** MQTT
